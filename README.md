# 📌 Overview

Moodify is an intelligent music recommendation platform that understands user emotions and suggests music accordingly.

The system uses:
- NLP-based sentiment analysis
- Speech recognition using Whisper
- Audio feature extraction with Librosa
- Spotify API integration

to create a seamless and personalized music experience.

# 🧠 How It Works

## 1️⃣ Text Mood Analysis
Users enter text describing their emotions.

Example:
> "I feel stressed and tired."

The system analyzes sentiment using NLP techniques.

---

## 2️⃣ Voice Recognition
Voice input is converted into text using OpenAI Whisper.

---

## 3️⃣ Audio Feature Extraction
Librosa extracts:
- Pitch
- Tempo
- Energy
- Frequency Patterns

These features help identify emotional tone.

---

## 4️⃣ Music Recommendation
Based on detected mood, Moodify recommends matching Spotify playlists.

| Mood | Recommendation |
|------|----------------|
| Happy | Upbeat Pop |
| Sad | Calm Acoustic |
| Energetic | EDM / Workout |
| Relaxed | Chill Lo-fi |
